# שיעור 4: איך לדבר עם AI - חלק ב': שלב מתקדם

## 🎯 מה תלמד בשיעור הזה

- טכניקות מתקדמות שישדרגו את התוצאות שלכם פי 10
- איך לבנות "שרשראות" פקודות לפרויקטים מורכבים
- מאגר Prompts מוכנים שתוכלו להעתיק ולהשתמש מיד

---

## 📖 מוכנים לשלב הבא?

בשיעור הקודם למדנו את היסודות - להיות ספציפיים, לתת הקשר, ולהגדיר פורמט. עכשיו נעלה הילוך. הטכניקות שנלמד עכשיו הן מה שמבדיל בין מישהו שמשתמש ב-AI לבין מישהו ש*שולט* ב-AI.

---

## טכניקה #1: שרשרת מחשבה (Chain of Thought) 🔗

במקום לבקש מ-AI תשובה סופית, בקשו ממנו **לחשוב שלב אחרי שלב**.

**❌ בלי שרשרת:**
```
כמה עולה לי לנסוע לעבודה בחודש?
```

**✅ עם שרשרת:**
```
אני גר בראשון לציון ועובד בתל אביב.
אני נוסע ברכב, 25 ק"מ לכיוון.
חשב לי שלב אחרי שלב:
1. כמה ק"מ בחודש (22 ימי עבודה, הלוך וחזור)
2. כמה ליטרים דלק (הנח צריכה ממוצעת של 8 ליטר ל-100 ק"מ)
3. כמה עולה הדלק (לפי מחיר עדכני)
4. תוסיף חניה (25 ש"ח ליום)
5. סה"כ חודשי
ואז תגיד לי: האם שווה לי לעבור לרכבת?
```

**למה זה עובד?** כי AI עושה פחות טעויות כשהוא "חושב בקול רם". במקום לקפוץ ישר לתשובה, הוא עובר שלב-שלב ובודק את עצמו.

---

## טכניקה #2: Prompting מותנה (If/Then) 🔀

אפשר לתת ל-AI הוראות שתלויות במצב:

```
אני הולך לשאול אותך שאלות על תזונה.

אם השאלה היא על אוכל בריא - ענה בקצרה ב-3 נקודות.
אם השאלה היא על מתכון - תן מתכון מלא עם מרכיבים וזמני הכנה.
אם השאלה היא רפואית - אל תענה, ותגיד לי לפנות לרופא.

מוכן? הנה השאלה הראשונה:
```

---

## טכניקה #3: מסגרת RICE לפתרון בעיות 🍚

שיטה מצוינת כשאתם צריכים ש-AI יעזור לכם לקבל החלטה:

```
אני שוקל [החלטה].

תנתח את זה לפי מסגרת RICE:
- Reach (טווח): כמה אנשים זה ישפיע עליהם?
- Impact (השפעה): כמה חזקה ההשפעה?
- Confidence (ביטחון): כמה אנחנו בטוחים בהערכות?
- Effort (מאמץ): כמה זמן/כסף/אנרגיה זה דורש?

תן ציון 1-10 לכל קטגוריה ותסביר למה.
```

---

## טכניקה #4: Prompt רב-שלבי (שרשרת שיחה) 💬

במקום לדחוף הכל ל-Prompt אחד ענק, פרקו את המשימה לשלבים:

**שלב 1 - מחקר:**
```
אני רוצה לפתוח עסק של משלוחי אוכל ביתי.
מהם 5 הדברים הכי חשובים שצריך לבדוק לפני שמתחילים?
```

**שלב 2 - העמקה:**
```
מצוין. בוא נעמיק בנקודה 2 - הרגולציה.
מה החוקים בישראל לגבי מכירת אוכל ביתי?
מה צריך רישיון עסק? אישור בריאות?
```

**שלב 3 - תוכנית:**
```
עכשיו, על סמך כל מה שדיברנו, תכין לי תוכנית פעולה
לשלושת החודשים הראשונים. חודש-חודש, עם משימות ספציפיות.
```

**למה זה עדיף על Prompt אחד ארוך?** כי בכל שלב אתם יכולים לכוון, לתקן, ולהעמיק. זה כמו שיחה עם יועץ - לא אומרים הכל בפעם אחת.

---

## טכניקה #5: בקשו ביקורת עצמית 🪞

טריק מעולה - בקשו מ-AI לבקר את עצמו:

```
תכתוב לי מייל שיווקי ללקוחות על מבצע קיץ.

אחרי שתכתוב, תבקר את עצמך:
- מה חזק במייל?
- מה חלש?
- תשכתב גרסה משופרת על סמך הביקורת
```

זה נשמע מוזר, אבל זה עובד מדהים. AI שמבקר את עצמו מייצר תוצאות טובות יותר.

---

## טכניקה #6: מגדירים את מי ש-AI לא צריך להיות 🚫

לפעמים, קל יותר להגיד מה אתם *לא* רוצים:

```
תכתוב לי פוסט על ביטוח בריאות.

אל תכתוב כמו:
- סוכן ביטוח שמנסה למכור
- אקדמאי משעמם
- רובוט בלי נשמה

כן תכתוב כמו:
- חבר טוב שמסביר דברים חשובים בפשטות
- מישהו שלא מפחד להגיד "זה מבלבל, אני יודע"
```

## מאגר Prompts מתקדמים - מוכנים להעתקה

### 🏢 Prompt לעסקים - ניתוח מתחרים
```
אני מנהל/ת [סוג עסק] ב[עיר].
המתחרים העיקריים שלי הם [שם 1] ו[שם 2].

תנתח עבורי:
1. מה הם עושים טוב שאני יכול ללמוד מהם?
2. מה חסר להם שאני יכול למלא?
3. 3 רעיונות להבדל תחרותי
4. תציע משפט שיווקי (slogan) שמדגיש את הייחוד שלי

תפורמט כטבלה + המלצות.
```

### 📧 Prompt לאימייל מושלם - כל סוג
```
תכתוב לי מייל:
- סוג: [בקשה / תלונה / תודה / הצעה / מעקב]
- נמען: [בוס / לקוח / ספק / קולגה]
- טון: [מקצועי / חמים / תקיף / מנומס]
- אורך: [3 שורות / 5 שורות / פסקה]
- נקודה עיקרית: [מה אני רוצה להשיג]
- מגבלה: [מה לא להגיד]
```

### 📚 Prompt ללימוד נושא חדש
```
אני רוצה ללמוד [נושא] מאפס.
רמת הידע הנוכחית שלי: [מתחיל / יש לי בסיס / מתקדם]
הזמן שיש לי: [שעה / שבוע / חודש]

בבקשה:
1. תן לי "מפת דרכים" - מה ללמוד ובאיזה סדר
2. 3 משאבים מומלצים (חינמיים)
3. פרויקט מעשי קטן שאוכל לעשות כדי לתרגל
4. הטעויות הנפוצות של מתחילים - ואיך להימנע מהן
```

### ✍️ Prompt לשכתוב טקסט
```
הנה טקסט שכתבתי:
[הדביקו טקסט]

שכתב אותו כך:
- שפה: [פורמלית / עממית / שיווקית / אקדמית]
- אורך: [קצר יותר ב-50% / זהה / ארוך יותר]
- קהל יעד: [מנהלים / צעירים / הורים / לקוחות]
- שמור על: [הנקודה המרכזית / הטון שלי / המבנה]
- שנה: [את הפתיחה / את הסיום / את הכל]
```

### 🎯 Prompt לסיעור מוחות
```
אני צריך רעיונות ל[נושא].

תן לי 10 רעיונות:
- 3 רעיונות "בטוחים" (קלאסיים, מוכחים)
- 3 רעיונות "יצירתיים" (שונים, מפתיעים)
- 3 רעיונות "מטורפים" (חוצפנים, לא שגרתיים)
- 1 רעיון ש"אף אחד לא חשב עליו"

לכל רעיון: שורה אחת של תיאור + הערכת קושי (קל/בינוני/קשה)
```

## 5 טעויות נפוצות (ואיך לתקן אותן)

### טעות #1: "תודה, AI" - להתייחס אליו כמו בן אדם
לא צריך לומר "בבקשה" ו"תודה". חסכו תווים ותהיו ישירים. AI לא נעלב.

### טעות #2: Prompt של 5 מילים על משימה מורכבת
"תכתוב לי עבודה" זה כמו להגיד לאדריכל "תבנה לי בית". בלי פרטים, התוצאה תהיה גנרית.

### טעות #3: לא להמשיך שיחה
הרבה אנשים כותבים Prompt אחד, מקבלים תשובה, ומתחילים שיחה חדשה. **המשיכו לדייק!** "טוב, אבל תקצר" / "תשנה את הטון" / "תוסיף דוגמה".

### טעות #4: לסמוך על AI בעיניים עצומות
AI ממציא דברים. תמיד בדקו עובדות חשובות, במיוחד מספרים, תאריכים, ושמות.

### טעות #5: לכתוב Prompt באנגלית כשרוצים תשובה בעברית
אם רוצים עברית - כתבו בעברית. זה עוזר ל-AI להבין שכל התשובה צריכה להיות בעברית.

## 💡 דוגמה מעשית - שיחה שלמה עם AI

הנה דוגמה לאיך שיחה מתקדמת עם AI צריכה להיראות:

**שלב 1:**
```
אני מורה היסטוריה בתיכון. אני צריך ליצור מבחן על מלחמת העולם השנייה
לכיתה י'. 20 שאלות.
```

**שלב 2** (אחרי שקיבלתם מבחן):
```
טוב, אבל:
1. יש פה יותר מדי שאלות על תאריכים - תחליף 5 מהן בשאלות הבנה
2. תוסיף 2 שאלות על השואה מנקודת מבט ישראלית
3. תעשה את שאלה 15 יותר קשה - היא קלה מדי
```

**שלב 3:**
```
מעולה. עכשיו תכין דף תשובות למורה, עם הסבר קצר לכל תשובה.
ותכין גם טבלת ציונים: 20 שאלות, 5 נקודות כל שאלה.
```

**רואים את הקסם?** שלושה שלבים, כל אחד מכוון ומדייק. התוצאה הסופית הרבה יותר טובה מ-Prompt אחד ארוך.

## 🧪 תרגיל לבית

**אתגר ה-Prompt המושלם:**

בחרו אחד מהתרחישים הבאים וכתבו את ה-Prompt הכי טוב שאתם יכולים, עם כל הטכניקות שלמדתם:

**תרחיש א':** אתם רוצים לתכנן חופשה משפחתית ביוון ל-5 ימים

**תרחיש ב':** אתם צריכים להכין מצגת לעבודה על "למה הצוות שלנו צריך תקציב נוסף"

**תרחיש ג':** אתם רוצים ללמוד לנגן על גיטרה ומחפשים תוכנית לימודים

**כללי המשחק:**
1. השתמשו בנוסחה: תפקיד + משימה + הקשר + פורמט + מגבלות
2. הוסיפו לפחות טכניקה אחת מתקדמת (שרשרת מחשבה / דוגמאות / ביקורת עצמית)
3. נסו את ה-Prompt ואז שפרו אותו לפחות פעם אחת

## 🎮 שאלה לבדיקה עצמית

**מה הטכניקה הכי טובה כשרוצים ש-AI יפתור בעיה מתמטית או לוגית?**

א. Few-shot - לתת דוגמאות של בעיות דומות

ב. Chain of Thought - לבקש ממנו לחשוב שלב אחרי שלב 🎯

ג. Role Playing - להגיד לו שהוא מתמטיקאי

ד. לכתוב את השאלה באנגלית

**התשובה הנכונה: ב'** ✅

**Chain of Thought (שרשרת מחשבה)** היא הטכניקה הכי אפקטיבית למשימות שדורשות חשיבה לוגית או חישובים. כשמבקשים מ-AI "חשוב שלב אחרי שלב", הוא עושה פחות טעויות כי הוא לא קופץ ישר לתשובה. זה דומה לתלמיד שמראה את דרך הפתרון במבחן - אם יש טעות, היא בדרך כלל קטנה יותר.

## 📌 סיכום הנקודות המרכזיות

- **שרשרת מחשבה** - בקשו מ-AI לחשוב שלב-שלב. פחות טעויות, יותר דיוק
- **שיחה רב-שלבית** - פרקו משימות גדולות לכמה Prompts קטנים
- **ביקורת עצמית** - בקשו מ-AI לבקר ולשפר את עצמו
- **הגדירו מה לא** - לפעמים קל יותר לומר מה אתם לא רוצים
- **5 הטעויות הנפוצות** - היו ספציפיים, המשיכו שיחות, ולא סומכים בעיניים עצומות
