- Готовые промпты для любой офисной задачи — письма, презентации, резюме

## 📖 Давайте поговорим по делу

Знаете этот момент, когда сидите перед пустым экраном в Word и нужно написать важное письмо? Или когда начальник просит «подготовь презентацию к завтра», а вы понятия не имеете, с чего начать?

**ИИ — это коллега, о котором вы всегда мечтали.** Он не ходит на обед, не жалуется и не сплетничает о начальстве.

Давайте посмотрим, как именно его использовать.

## 📧 Письма — главная боль каждого

### Профессиональное письмо — универсальный шаблон
```
Напиши профессиональное письмо на русском:

Получатель: [кто — начальник/клиент/поставщик/коллега]
Тема: [о чём письмо]
Цель: [что я хочу, чтобы произошло после прочтения]
Тон: [вежливый/твёрдый/дружелюбный/формальный]
Длина: [короткое 3-4 строки / среднее / подробное]
Важные детали: [всё, что нужно включить]

Не забудь: чёткую тему письма, обращение, тело и вежливое завершение.
```

### Быстрый ответ на письмо
```
Я получил следующее письмо:
[вставьте письмо]

Напиши ответ, который:
- Благодарит за обращение
- Отвечает на основной вопрос
- Предлагает следующий шаг
- Короткий (максимум 5 строк)
```

### Вежливая, но эффективная жалоба
```
Мне нужно написать письмо-жалобу в [компания/поставщик] по поводу [проблема].

Письмо должно:
- Начаться позитивно (обычно я довольный клиент)
- Описать проблему фактически (что случилось, когда, номер заказа)
- Объяснить последствия для меня
- Попросить конкретное решение
- Быть твёрдым, но уважительным (без угроз, без крика)

5-7 строк максимум.
```

## 📊 Презентации — от нуля до готовой презентации

### Шаг 1: Структура
```
Мне нужна презентация на тему [тема] для [аудитория — начальник/команда/клиенты/инвесторы].
Длительность выступления: [10/20/30 минут]

Построй структуру:
- Сколько слайдов рекомендуется
- Заголовок каждого слайда
- 3-4 ключевых пункта для каждого слайда
- Предложение по графике/изображению для каждого слайда
```

### Шаг 2: Контент
```
Теперь напиши полный контент для слайда [номер]: [заголовок].
Напиши:
- Сильный заголовок
- 3-4 буллет-поинта (каждый до 10 слов)
- Заметки спикера — что сказать устно по этому слайду (2-3 предложения)
```

### Шаг 3: Открытие и финал
```
Напиши мне:
1. Впечатляющее открытие презентации — фраза, которая захватит внимание
2. Сильный финал — резюме + призыв к действию
3. Ожидаемые вопросы от аудитории + готовые ответы (3 вопроса)
```

## 📝 Резюме — от длинных совещаний к ключевым пунктам

### Резюме совещания
```
Вот стенограмма/заметки с совещания:
[вставьте стенограмму или заметки]

Подготовь резюме, включающее:
1. Тема совещания и участники
2. 5 ключевых моментов, которые обсуждались
3. Принятые решения
4. Задачи — кто, что, когда (таблица)
5. Открытые вопросы для следующего совещания
```

### Резюме длинного документа
```
Вот документ [тип — договор/отчёт/политика]:
[вставьте или загрузите файл]

Обобщи его:
- Краткое резюме для руководства (3 строки)
- Основные пункты (5-7 буллетов)
- Моменты, требующие особого внимания
- Вопросы, которые стоит задать
```

## 📈 Анализ данных — Excel теперь менее страшен с ИИ

### Анализ данных продаж
```
Вот наши данные по продажам за [период]:
[вставьте данные или опишите их]

Проанализируй для меня:
1. Основные тренды — что растёт, что падает
2. Самый прибыльный продукт/услуга
3. Сезонность — есть ли сильные/слабые месяцы
4. 3 практические рекомендации по улучшению
5. Дай формулы для Excel, которые мне нужны для отслеживания
```

### Создание формул Excel
```
Мне нужна формула в Excel, которая:
[опишите, что формула должна делать]

Мои данные:
- Столбец A: [что там]
- Столбец B: [что там]
- Столбец C: [что там]

Дай мне:
1. Формулу
2. Краткое объяснение, что она делает
3. Числовой пример
```

## ✍️ Профессиональное письмо

### Коммерческое предложение / Предложение проекта
```
Мне нужно написать предложение для [клиент/компания] на [услуга/продукт].

Детали:
- Что я предлагаю: [описание]
- Цена: [сумма]
- Сроки: [сколько времени]
- Моё преимущество: [почему именно я]

Напиши профессиональное предложение, включающее:
1. Вступление, показывающее, что я понимаю их потребности
2. Описание моего решения
3. Преимущества (3-4 пункта)
4. Условия и цена
5. Призыв к действию
```

### Профессиональный пост для LinkedIn
```
Напиши пост для LinkedIn на тему [тема].

Я [должность] в [сфере] с [X лет опыта].

Пост должен:
- Начаться с хука (первое предложение, которое цепляет)
- Поделиться реальным инсайтом из моего опыта
- Закончиться вопросом, приглашающим к дискуссии
- Длина: 5-8 строк
- Тон: профессиональный, но человечный (не «роботный»)
- 3-5 релевантных хэштегов
```

## 💡 Практический пример — рабочий день с ИИ

Давайте посмотрим, как обычный рабочий день выглядит с ИИ:

**08:30 — Разбор почты**
```
Вот 5 писем, которые я получил утром:
[вставьте письма]

Расставь приоритеты от 1 (срочно) до 5 (может подождать).
Для каждого письма: одна строка — что нужно сделать и когда.
```

**10:00 — Подготовка к совещанию**
```
У меня через час совещание с [клиент/команда] по [тема].
Подготовь мне:
1. 3 ключевых пункта, которые стоит поднять
2. 2 умных вопроса для обсуждения
3. Резюме в 3 предложениях по теме (быстрое освежение)
```

**14:00 — Написание отчёта**
```
Помоги мне написать еженедельный отчёт для руководителя.
Вот что я сделал за неделю:
[список действий]

Оформи как профессиональный отчёт с:
- Заголовком
- Достижения недели (буллеты)
- Проблемы и решения
- План на следующую неделю
```

## 📌 Резюме ключевых моментов

- **Письма:** Дайте ИИ получателя + цель + тон + длину = идеальное письмо за секунды
- **Презентации:** Работайте поэтапно — структура → контент → открытие и финал
- **Резюме:** ИИ отлично превращает длинные совещания и документы в ключевые пункты
- **Данные:** ИИ может анализировать и писать формулы Excel за вас
- **Правило:** Начните использовать ИИ для одной *реальной* задачи сегодня
