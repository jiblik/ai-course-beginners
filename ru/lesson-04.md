## Техника #1: Цепочка мыслей (Chain of Thought) 🔗

Вместо того чтобы просить у ИИ финальный ответ, попросите его **думать шаг за шагом**.

**❌ Без цепочки:**
```
Сколько мне стоит ездить на работу в месяц?
```

**✅ С цепочкой:**
```
Я живу в пригороде и работаю в центре города.
Я езжу на машине, 30 км в одну сторону.
Рассчитай шаг за шагом:
1. Сколько км в месяц (22 рабочих дня, туда и обратно)
2. Сколько литров бензина (расход 9 литров на 100 км)
3. Сколько стоит бензин (по актуальной цене в моей стране)
4. Добавь парковку ($10 в день)
5. Итого в месяц
А потом скажи: стоит ли мне пересесть на метро?
```

**Почему это работает?** Потому что ИИ делает меньше ошибок, когда «думает вслух». Вместо того чтобы сразу прыгнуть к ответу, он проходит шаг за шагом и проверяет себя.

## Техника #2: Условный Prompting (If/Then) 🔀

Можно дать ИИ инструкции, зависящие от ситуации:

```
Я буду задавать тебе вопросы о питании.

Если вопрос о здоровой еде — отвечай кратко, 3 пункта.
Если вопрос о рецепте — дай полный рецепт с ингредиентами и временем приготовления.
Если вопрос медицинский — не отвечай и скажи обратиться к врачу.

Готов? Вот первый вопрос:
```

## Техника #3: Фреймворк RICE для решения проблем 🍚

Отличный метод, когда нужна помощь ИИ в принятии решения:

```
Я рассматриваю [решение].

Проанализируй по фреймворку RICE:
- Reach (охват): на скольких людей это повлияет?
- Impact (влияние): насколько сильно влияние?
- Confidence (уверенность): насколько мы уверены в оценках?
- Effort (усилия): сколько времени/денег/энергии потребуется?

Дай оценку 1-10 по каждой категории и объясни почему.
```

## Техника #4: Многоэтапный промпт (цепочка диалога) 💬

Вместо того чтобы впихнуть всё в один гигантский промпт, разбейте задачу на этапы:

**Этап 1 — Исследование:**
```
Я хочу открыть бизнес по доставке домашней еды.
Какие 5 самых важных вещей нужно проверить перед стартом?
```

**Этап 2 — Углубление:**
```
Отлично. Давай углубимся в пункт 2 — регулирование.
Какие законы в моей стране касательно продажи домашней еды?
Нужна ли лицензия? Санитарные разрешения?
```

**Этап 3 — План:**
```
Теперь, на основе всего, что мы обсудили, составь план действий
на первые три месяца. Помесячно, с конкретными задачами.
```

**Почему это лучше одного длинного промпта?** Потому что на каждом этапе вы можете корректировать, поправлять и углублять. Это как разговор с консультантом — не говорите всё сразу.

## Техника #5: Попросите самокритику 🪞

Крутой трюк — попросите ИИ покритиковать самого себя:

```
Напиши маркетинговое письмо клиентам о летней распродаже.

После того как напишешь, покритикуй себя:
- Что сильного в письме?
- Что слабого?
- Перепиши улучшенную версию на основе критики
```

Звучит странно, но работает потрясающе. ИИ, критикующий себя, выдаёт лучшие результаты.

---

## Техника #6: Определите, кем ИИ НЕ должен быть 🚫

Иногда проще сказать, чего вы *не* хотите:

```
Напиши пост о медицинском страховании.

Не пиши как:
- Страховой агент, который пытается продать
- Скучный академик
- Робот без души

Пиши как:
- Хороший друг, который объясняет важные вещи просто
- Человек, который не боится сказать "да, это запутано, я знаю"
```

## Банк продвинутых промптов — готовы к копированию

### 🏢 Промпт для бизнеса — Анализ конкурентов
```
Я управляю [тип бизнеса] в [город].
Мои основные конкуренты — [название 1] и [название 2].

Проанализируй для меня:
1. Что они делают хорошо, чему я могу научиться?
2. Чего им не хватает, что я могу заполнить?
3. 3 идеи для конкурентного преимущества
4. Предложи маркетинговый слоган, подчёркивающий мою уникальность

Оформи как таблицу + рекомендации.
```

### 📧 Промпт для идеального email — любой тип
```
Напиши мне письмо:
- Тип: [просьба / жалоба / благодарность / предложение / напоминание]
- Получатель: [начальник / клиент / поставщик / коллега]
- Тон: [профессиональный / тёплый / твёрдый / вежливый]
- Длина: [3 строки / 5 строк / абзац]
- Главная цель: [чего я хочу добиться]
- Ограничение: [что не говорить]
```

### 📚 Промпт для изучения новой темы
```
Я хочу изучить [тему] с нуля.
Мой текущий уровень знаний: [начинающий / есть база / продвинутый]
Имеющееся время: [час / неделя / месяц]

Пожалуйста:
1. Дай «дорожную карту» — что изучать и в каком порядке
2. 3 рекомендуемых ресурса (бесплатных)
3. Небольшой практический проект для тренировки
4. Типичные ошибки новичков — и как их избежать
```

### ✍️ Промпт для переписывания текста
```
Вот текст, который я написал:
[вставьте текст]

Перепиши его так:
- Язык: [формальный / разговорный / маркетинговый / академический]
- Длина: [короче на 50% / такая же / длиннее]
- Целевая аудитория: [руководители / молодёжь / родители / клиенты]
- Сохрани: [основную мысль / мой тон / структуру]
- Измени: [вступление / концовку / всё]
```

### 🎯 Промпт для мозгового штурма
```
Мне нужны идеи для [тема].

Дай мне 10 идей:
- 3 «безопасные» (классические, проверенные)
- 3 «креативные» (необычные, удивительные)
- 3 «безумные» (дерзкие, нестандартные)
- 1 идея, «о которой никто не подумал»

Для каждой: одна строка описания + оценка сложности (легко/средне/сложно)
```

## 5 частых ошибок (и как их исправить)

### Ошибка #1: «Спасибо, ИИ» — относиться к нему как к человеку
Не нужно говорить «пожалуйста» и «спасибо». Экономьте символы и будьте прямыми. ИИ не обижается.

### Ошибка #2: Промпт из 5 слов для сложной задачи
«Напиши мне работу» — это как сказать архитектору «построй мне дом». Без деталей результат будет generic.

### Ошибка #3: Не продолжать диалог
Многие пишут один промпт, получают ответ и начинают новый чат. **Продолжайте уточнять!** «Хорошо, но сократи» / «Измени тон» / «Добавь пример».

### Ошибка #4: Слепо доверять ИИ
ИИ выдумывает вещи. Всегда проверяйте важные факты, особенно числа, даты и имена.

### Ошибка #5: Писать промпт на английском, когда нужен ответ на русском
Если хотите русский — пишите на русском. Это помогает ИИ понять, что весь ответ должен быть на русском.

## 💡 Практический пример — полный диалог с ИИ

Вот как продвинутый разговор с ИИ должен выглядеть:

**Этап 1:**
```
Я учитель истории в школе. Мне нужно создать контрольную по Второй мировой войне
для 10 класса. 20 вопросов.
```

**Этап 2** (после получения контрольной):
```
Хорошо, но:
1. Слишком много вопросов на даты — замени 5 из них на вопросы на понимание
2. Добавь 2 вопроса о Великой Отечественной войне
3. Сделай вопрос 15 сложнее — он слишком лёгкий
```

**Этап 3:**
```
Отлично. Теперь подготовь лист ответов для учителя с кратким объяснением к каждому ответу.
И таблицу оценок: 20 вопросов, 5 баллов за каждый.
```

**Видите магию?** Три этапа, каждый корректирует и уточняет. Финальный результат гораздо лучше, чем от одного длинного промпта.

## 📌 Резюме ключевых моментов

- **Цепочка мыслей** — просите ИИ думать пошагово. Меньше ошибок, больше точности
- **Многоэтапный диалог** — разбивайте большие задачи на несколько маленьких промптов
- **Самокритика** — просите ИИ критиковать и улучшать самого себя
- **Определите «что нет»** — иногда проще сказать, чего вы не хотите
- **5 частых ошибок** — будьте конкретными, продолжайте диалоги, не доверяйте слепо
