## 🎯 Что вы узнаете в этом уроке

- Что такое промпт и почему от него зависит, получите вы отличный ответ или мусор
- 5 золотых правил написания хорошего промпта
- Практические примеры, которые можно копировать и использовать прямо сейчас

## 📖 Почему «разговаривать с ИИ» — это искусство?

Начнём с аналогии:

Представьте, что вы пришли в ресторан и сказали официанту: **«Принеси мне еду.»**

Что вы получите? Кто знает. Может, пиццу, может, салат, может, рыбу. Официант принесёт *что-то*, но скорее всего, не совсем то, что вы хотели.

А теперь представьте, что вы сказали: **«Я хочу стейк medium rare, с запечённым картофелем, перечным соусом, без лука, и ещё бокал красного вина.»**

Гораздо лучше, правда?

**Так же и с ИИ.** То, что вы ему пишете (**промпт**) определяет, что вы получите в ответ. Размытый промпт → размытый ответ. Точный промпт → точный ответ.

---

## Что такое промпт?

**Промпт (Prompt)** = текст, который вы пишете ИИ. Это инструкция, вопрос, запрос — всё, что вы набираете в текстовом поле.

Примеры промптов:
- «Какая погода?» ← Простой промпт
- «Напиши мне письмо» ← Размытый промпт
- «Напиши мне деловое письмо на русском моей начальнице с просьбой о выходном в четверг, в вежливом, но не подобострастном тоне» ← Отличный промпт

**Простое правило:** Чем больше информации вы дадите ИИ о том, что хотите, тем лучше будет ответ.

## 5 золотых правил написания промпта

### Правило #1: Будьте конкретными 🎯

| ❌ Слабый промпт | ✅ Сильный промпт |
|-----------------|------------------|
| «Напиши что-нибудь про маркетинг» | «Напиши 5 идей для постов в Instagram для маленького цветочного магазина, сфокусированных на весеннем сезоне» |
| «Помоги с письмом» | «Напиши короткое письмо (5-6 строк) поставщику, который не доставил заказ вовремя, в профессиональном, но твёрдом тоне» |
| «Расскажи про готовку» | «Объясни, как приготовить по-настоящему вкусный домашний борщ, шаг за шагом, для новичка, который никогда не готовил» |

### Правило #2: Давайте контекст 📋

ИИ не знает, кто вы, чем занимаетесь и почему спрашиваете. Расскажите ему!

**❌ Без контекста:**
```
Напиши мне био для Instagram
```

**✅ С контекстом:**
```
Я графический дизайнер-фрилансер, специализируюсь на создании логотипов и брендинга для малого бизнеса.
Моя целевая аудитория — владельцы малого бизнеса 30-50 лет.
Напиши мне био для Instagram, профессиональное, но тёплое, до 150 символов.
```

### Правило #3: Задайте формат 📐

Скажите ИИ, в каком виде хотите получить ответ:

```
Ответь мне в виде:
- Списка пунктов (не длинных абзацев)
- Максимум 5 пунктов
- Каждый пункт — одно предложение
```

Или:
```
Ответь в формате таблицы с тремя колонками: действие, время, стоимость
```

Или:
```
Отвечай, как будто объясняешь 8-летнему ребёнку
```

### Правило #4: Давайте примеры 🎨

Это одна из самых мощных техник. Если хотите результат в определённом стиле — покажите ИИ пример.

```
Напиши описания товаров для интернет-магазина в таком стиле:

Пример: "Эта кепка? Это не просто кепка. Это ваше заявление миру, что вы пришли.
100% хлопок. 100% стиль. 0% извинений."

Теперь напиши в таком же стиле для:
1. Чёрная футболка
2. Рюкзак для работы
3. Термобутылка
```

### Правило #5: Возвращайтесь и улучшайте 🔄

Не получили то, что хотели? Не начинайте заново. **Продолжайте диалог:**

```
Пример диалога:

Вы: "Напиши письмо поставщику"
ИИ: [пишет длинное и слишком формальное письмо]
Вы: "Хорошо, но слишком длинно. Сократи вдвое и будь более прямым"
ИИ: [пишет более короткую версию]
Вы: "Отлично, только добавь крайний срок для ответа"
ИИ: [добавляет]
```

**ИИ — как талантливый сотрудник, которому нужно направление.** Если первый результат не идеален — это нормально. Корректируйте, направляйте, уточняйте.

## Секретная формула идеального промпта

Вот шаблон, который работает почти всегда:

```
[Роль] + [Задача] + [Контекст] + [Формат] + [Ограничения]
```

**Разобранный пример:**

```
Ты [опытный копирайтер на русском языке].               ← Роль
Напиши [пост для Facebook, продвигающий кулинарный мастер-класс]. ← Задача
Мастер-класс [для пар, стоит $50, в эту субботу,           ← Контекст
фокус на итальянскую кухню].
Пост должен быть [3-4 строки, с эмодзи,                   ← Формат
включая призыв к действию].
[Не используй слова «уникальная возможность» или            ← Ограничения
«не упустите шанс»].
```

## Базовые техники, которые стоит знать

### 1. Zero-shot (выстрел с нуля) 🎯
Просто спрашиваете, без примеров:
```
Переведи на английский: "Я хочу забронировать столик на двоих на вечер"
```

### 2. Few-shot (несколько примеров) 🎯🎯🎯
Даёте примеры перед запросом:
```
Переведи на разговорный английский (не формальный):
Русский: "Как дела, братан?" → Английский: "What's up bro?"
Русский: "Это было безумие" → Английский: "That was insane"
Русский: "Я без ума от этой пиццы" → Английский: ???
```

### 3. Метод «Роли» (Role Playing) 🎭
```
Ты учитель экономики в школе. Объясни мне, что такое инфляция
простым языком с примерами из повседневной жизни.
```

### 4. Метод «Шаг за шагом» (Step by Step) 👣
```
Я хочу открыть YouTube-канал.
Дай мне план действий шаг за шагом:
1. Что сделать в первый день
2. Что сделать в первую неделю
3. Что сделать в первый месяц
```

## 💡 Практические примеры — готовые промпты

### Промпт #1: Помощь с резюме
```
Я хочу улучшить своё резюме.
Я [дизайнер/программист/бухгалтер/менеджер проектов] с [X] годами опыта.
Пожалуйста:
1. Предложи 3 сильных предложения для вступительной секции (Summary)
2. Помоги сформулировать мой опыт в [название компании], подчёркивая достижения, а не обязанности
3. Используй сильные глаголы действия
```

### Промпт #2: Планирование на неделю
```
Составь меню ужинов на неделю:
- Для семьи из 4 человек (2 взрослых, 2 ребёнка 6 и 9 лет)
- Бюджет: средний
- Ограничения: без морепродуктов, младший не любит грибы
- Только ужины (обычная домашняя еда)
- Добавь список покупок в магазин на всю неделю
```

### Промпт #3: Подготовка к собеседованию
```
Я иду на собеседование на должность [название] в [тип компании].
1. Дай мне 10 частых вопросов, которые скорее всего зададут
2. К каждому вопросу — краткий совет, как отвечать
3. Дай 3 умных вопроса, которые я могу задать интервьюеру
```

### Промпт #4: Объяснение сложного понятия
```
Объясни мне, что такое [блокчейн / NFT / акции / страхование жизни],
как будто мне 12 лет. Без профессиональных терминов.
Используй пример из повседневной жизни.
Максимум 10 строк.
```

### Промпт #5: Критика текста
```
Вот текст, который я написал [вставьте текст].
Пожалуйста:
1. Укажи 3 хорошие стороны текста
2. Укажи 3 вещи, которые можно улучшить
3. Перепиши первый абзац как пример улучшения
Не меняй мой стиль — только улучши.
```

## 🧪 Домашнее задание

**Часть 1: Формула**
Возьмите секретную формулу:
```
[Роль] + [Задача] + [Контекст] + [Формат] + [Ограничения]
```
и напишите свой промпт на интересующую вас тему. Попробуйте его в ИИ-инструменте.

**Часть 2: До и после**
Попробуйте оба промпта и сравните:

**Слабый промпт:**
```
Напиши мне про собак
```

**Сильный промпт:**
```
Я подумываю завести собаку впервые. Я живу в квартире на 3 этаже без лифта,
работаю 8 часов вне дома, и у меня ребёнок 4 лет.
Порекомендуй 3 подходящие породы и объясни, почему каждая из них подходит для моей ситуации.
```

Обратите внимание, насколько второй ответ полезнее!

## 🎮 Вопрос для самопроверки

**В чём разница между Zero-shot и Few-shot?**

а. Zero-shot — бесплатно, а Few-shot — платно

б. Zero-shot — спрашиваете напрямую, Few-shot — сначала даёте примеры 🎯

в. Zero-shot — для текста, а Few-shot — для изображений

г. Разницы нет, только название разное

<details>
<summary>Нажмите для ответа</summary>

**Правильный ответ: б** ✅

**Zero-shot** = спрашиваете ИИ напрямую без примеров («Переведи: привет» → «Hello»)
**Few-shot** = даёте несколько примеров перед запросом, чтобы ИИ понял нужный вам стиль. Это очень мощная техника, когда нужен результат в определённом стиле.

</details>

- **Формула:** Роль + Задача + Контекст + Формат + Ограничения
