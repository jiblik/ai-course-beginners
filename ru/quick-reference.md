## 📋 Формула идеального промпта

```
[Роль] + [Задача] + [Контекст] + [Формат] + [Ограничения]
```

**Пример:**
```
Ты [опытный копирайтер].                     ← Роль
Напиши [письмо клиенту].                     ← Задача
Клиент [не получил заказ, ждёт неделю].       ← Контекст
Письмо должно быть [5 строк, вежливое].      ← Формат
[Не предлагай денежную компенсацию].          ← Ограничения
```

## 🎯 5 золотых принципов

| # | Принцип | Пример |
|---|---|---|
| 1 | **Будьте конкретны** | «5 идей для поста о цветочном магазине» ← не «напиши про маркетинг» |
| 2 | **Давайте контекст** | «Я графический дизайнер-фрилансер из Москвы...» |
| 3 | **Определяйте формат** | «Ответь списком, максимум 5 пунктов» |
| 4 | **Давайте примеры** | «В таком стиле: [пример]» |
| 5 | **Итерируйте** | «Хорошо, но сократи и будь более прямым» |

## 🔧 Продвинутые техники — напоминание

| Техника | Когда использовать | Как |
|---|---|---|
| **Zero-shot** | Простые вопросы | Спрашиваете напрямую |
| **Few-shot** | Нужен конкретный стиль | Даёте 2-3 примера перед запросом |
| **Chain of Thought** | Расчёты/логика | «Думай шаг за шагом» |
| **Role Playing** | Нужна точка зрения | «Ты учитель/врач/шеф-повар...» |
| **Самокритика** | Нужен идеальный результат | «После написания — покритикуй себя» |

## 📧 Готовые шаблоны промптов

### Профессиональное письмо
```
Напиши письмо [тип] для [адресата] на тему [тема].
Тон: [вежливый/настойчивый/дружелюбный]. Длина: [X строк].
Цель: [что я хочу, чтобы произошло].
```

### Конспект документа
```
Сделай конспект этого документа:
- Краткое резюме (3 строки)
- Основные моменты (5 пунктов)
- На что обратить внимание
```

### Мозговой штурм
```
Дай 10 идей по [теме]:
3 безопасных, 3 креативных, 3 безумных, 1 такую, о которой никто не подумал.
```

### Изучение новой темы
```
Обучи меня [теме] с нуля.
После каждого объяснения задай вопрос для проверки понимания.
```

### Планирование
```
Спланируй мне [задачу/мероприятие/проект].
Детали: [X]. Бюджет: [X]. Срок: [X].
Оформи как таймлайн.
```

## ⚠️ Правила безопасности — напоминание

### Не делитесь:
- ❌ Паспортные данные, номера карт, пароли
- ❌ Секретные документы компании
- ❌ Чувствительная медицинская информация
- ❌ Личные данные других людей

### Всегда:
- ✅ Проверяйте важные факты
- ✅ Читайте перед отправкой
- ✅ Консультируйтесь со специалистом при серьёзных решениях
- ✅ Отключите обучение на ваших данных (в настройках)
