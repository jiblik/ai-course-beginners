# שיעור 5: AI בעבודה - להפוך לסופרמן של המשרד

## 🎯 מה תלמד בשיעור הזה

- איך AI יכול לחסוך לכם שעות עבודה כל שבוע
- Prompts מוכנים לכל משימה משרדית - מיילים, מצגות, סיכומים
- כלים ספציפיים שישדרגו את הפרודוקטיביות שלכם

---

## 📖 בואו נדבר תכל'ס

אתם יודעים את הרגע הזה שאתם יושבים מול מסך ריק ב-Word וצריכים לכתוב מייל חשוב? או כשהבוס מבקש "תכין מצגת עד מחר" ואין לכם מושג מאיפה להתחיל?

**AI הוא חבר העבודה שתמיד חלמתם עליו.** הוא לא הולך להפסקת צהריים, לא מתלונן, ולא עושה רכילות על הבוס.

בואו נראה בדיוק איך להשתמש בו.

---

## 📧 מיילים - הכאב הכי גדול של כולנו

### מייל פרופסיונלי בעברית - תבנית אוניברסלית
```
תכתוב מייל מקצועי בעברית:

נמען: [מי מקבל - בוס/לקוח/ספק/עמית]
נושא: [על מה המייל]
מטרה: [מה אני רוצה שיקרה אחרי שיקראו]
טון: [מנומס/תקיף/ידידותי/רשמי]
אורך: [קצר 3-4 שורות / בינוני / מפורט]
פרטים חשובים: [כל מה שצריך לכלול]

אל תשכח: שורת נושא ברורה, פנייה, גוף, וסיום מנומס.
```

### מייל תגובה מהירה
```
קיבלתי את המייל הבא:
[הדביקו את המייל]

תכתוב תגובה ש:
- מודה על הפנייה
- עונה על הנקודה העיקרית
- מציעה צעד הבא
- קצרה (מקסימום 5 שורות)
```

### מייל תלונה מנומסת אבל אפקטיבית
```
אני צריך לכתוב מייל תלונה ל[חברה/ספק] על [הבעיה].

המייל צריך:
- להתחיל חיובי (אני לקוח מרוצה בדרך כלל)
- לתאר את הבעיה בצורה עובדתית (מה קרה, מתי, מספרי הזמנה)
- להסביר את ההשפעה עלי
- לבקש פתרון ספציפי
- להיות תקיף אבל מכבד (לא איום, לא צעקות)

5-7 שורות מקסימום.
```

## 📊 מצגות - מאפס למצגת מוכנה

### שלב 1: מבנה
```
אני צריך מצגת על [נושא] ל[קהל - בוס/צוות/לקוחות/משקיעים].
משך ההצגה: [10/20/30 דקות]

תבנה לי מבנה:
- כמה שקופיות מומלצות
- כותרת לכל שקופית
- 3-4 נקודות עיקריות לכל שקופית
- הצעה לגרפיקה/תמונה בכל שקופית
```

### שלב 2: תוכן
```
עכשיו תכתוב את התוכן המלא לשקופית [מספר]: [כותרת].
כתוב:
- כותרת חזקה
- 3-4 bullets (כל אחד עד 10 מילים)
- הערת דובר - מה להגיד בעל פה על השקופית הזו (2-3 משפטים)
```

### שלב 3: פתיחה וסיום
```
תכתוב לי:
1. פתיחה מרשימה למצגת - משפט פתיחה שיתפוס תשומת לב
2. סיום חזק - סיכום + קריאה לפעולה
3. שאלות צפויות מהקהל + תשובות מוכנות (3 שאלות)
```

## 📝 סיכומים - מפגישות ארוכות לנקודות מפתח

### סיכום פגישה
```
הנה תמלול/הערות מפגישה:
[הדביקו תמלול או הערות]

תכין סיכום שכולל:
1. נושא הפגישה ומשתתפים
2. 5 נקודות מפתח שהועלו
3. החלטות שהתקבלו
4. משימות - מי עושה מה עד מתי (טבלה)
5. נושאים פתוחים לפגישה הבאה
```

### סיכום מסמך ארוך
```
הנה מסמך [סוג - חוזה/דו"ח/מדיניות]:
[הדביקו או העלו קובץ]

תסכם אותו:
- תקציר מנהלים (3 שורות)
- נקודות עיקריות (5-7 bullets)
- דברים שדורשים תשומת לב מיוחדת
- שאלות שכדאי לשאול
```

## 📈 ניתוח נתונים - Excel זה פחות מפחיד עם AI

### ניתוח נתוני מכירות
```
הנה נתוני המכירות שלנו ל-[תקופה]:
[הדביקו נתונים או תארו אותם]

תנתח עבורי:
1. מגמות עיקריות - מה עולה, מה יורד
2. המוצר/שירות הכי רווחי
3. עונתיות - האם יש חודשים חזקים/חלשים
4. 3 המלצות מעשיות לשיפור
5. תן לי את הנוסחאות ל-Excel שאני צריך כדי לעקוב אחרי זה
```

### יצירת נוסחאות Excel
```
אני צריך נוסחה ב-Excel ש:
[תארו מה הנוסחה צריכה לעשות]

הנתונים שלי:
- עמודה A: [מה יש שם]
- עמודה B: [מה יש שם]
- עמודה C: [מה יש שם]

תן לי:
1. את הנוסחה
2. הסבר קצר מה היא עושה
3. דוגמה מספרית
```

## ✍️ כתיבה מקצועית

### הצעת מחיר / הצעת פרויקט
```
אני צריך לכתוב הצעה ל[לקוח/חברה] עבור [שירות/מוצר].

פרטים:
- מה אני מציע: [תיאור]
- מחיר: [סכום]
- לוח זמנים: [כמה זמן]
- היתרון שלי: [למה דווקא אני]

תכתוב הצעה מקצועית שכוללת:
1. פתיחה שמראה שאני מבין את הצורך שלהם
2. תיאור הפתרון שלי
3. יתרונות (3-4 נקודות)
4. תנאים ומחיר
5. קריאה לפעולה
```

### פוסט לינקדאין מקצועי
```
תכתוב לי פוסט לינקדאין על [נושא].

אני [תפקיד] ב[תחום] עם [X שנות ניסיון].

הפוסט צריך:
- לפתוח עם הוק (משפט ראשון שתופס)
- לשתף תובנה אמיתית מהניסיון שלי
- לסיים עם שאלה שמזמינה שיחה
- אורך: 5-8 שורות
- טון: מקצועי אבל אנושי (לא "רובוטי")
- 3-5 האשטאגים רלוונטיים
```

## 💡 דוגמה מעשית - יום עבודה עם AI

בואו נראה איך יום עבודה רגיל נראה עם AI:

**08:30 - סידור המייל**
```
הנה 5 מיילים שקיבלתי הבוקר:
[הדביקו מיילים]

תעדף אותם מ-1 (דחוף) ל-5 (יכול לחכות).
לכל מייל: שורה אחת - מה צריך לעשות ומתי.
```

**10:00 - הכנה לפגישה**
```
יש לי פגישה בעוד שעה עם [לקוח/צוות] על [נושא].
תכין לי:
1. 3 נקודות מפתח שכדאי שאעלה
2. 2 שאלות חכמות לשאול
3. סיכום של 3 משפטים על הנושא (רענון מהיר)
```

**14:00 - כתיבת דו"ח**
```
תעזור לי לכתוב דו"ח שבועי למנהל.
הנה מה שעשיתי השבוע:
[רשימת פעילויות]

תפורמט כדו"ח מקצועי עם:
- כותרת
- הישגי השבוע (bullets)
- אתגרים ופתרונות
- תוכנית לשבוע הבא
```

## 🧪 תרגיל לבית

**בחרו אחת מהמשימות הבאות ועשו אותה באמת:**

**אפשרות א':** כתבו מייל מקצועי שצריכים לשלוח (באמת!) - השתמשו ב-AI וראו כמה זמן חסכתם.

**אפשרות ב':** קחו מסמך ארוך (חוזה, מדיניות, דו"ח) והעלו אותו ל-Claude. בקשו סיכום ונקודות עיקריות.

**אפשרות ג':** בקשו מ-AI לכתוב לכם מבנה למצגת על נושא רלוונטי לעבודה שלכם.

**הנקודה:** לא סתם לתרגל, אלא לעשות משהו **אמיתי** שחוסך לכם זמן.

## 🎮 שאלה לבדיקה עצמית

**מה הגישה הנכונה כשמבקשים מ-AI לכתוב מייל מקצועי?**

א. לכתוב "תכתוב מייל" ולסמוך על AI שיבין

ב. לציין נמען, מטרה, טון ואורך - ואז לדייק בהמשך השיחה 🎯

ג. לכתוב את המייל בעצמך ולבקש מ-AI רק לבדוק שגיאות

ד. להשתמש רק ב-Copilot כי הוא של מיקרוסופט

<details>
<summary>לחצו לתשובה</summary>

**התשובה הנכונה: ב'** ✅

הגישה הטובה ביותר היא לתת ל-AI את כל ההקשר - מי מקבל, מה המטרה, באיזה טון, ובאיזה אורך. ואם התוצאה לא מושלמת - לדייק בהמשך ("תקצר", "תהיה יותר ישיר", "תוסיף את הנתון הזה"). זה משלב את עקרונות ה-Prompting שלמדנו בשיעורים 3-4.

</details>

## 📌 סיכום הנקודות המרכזיות

- **מיילים:** תנו ל-AI נמען + מטרה + טון + אורך = מייל מושלם בשניות
- **מצגות:** עבדו בשלבים - מבנה → תוכן → פתיחה וסיום
- **סיכומים:** AI מצוין בלהפוך פגישות ומסמכים ארוכים לנקודות מפתח
- **נתונים:** AI יכול לנתח ולכתוב נוסחאות Excel בשבילכם
- **הכלל:** התחילו להשתמש ב-AI במשימה *אמיתית* אחת היום
