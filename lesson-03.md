# שיעור 3: איך לדבר עם AI - חלק א': היסודות

## 🎯 מה תלמד בשיעור הזה

- מה זה Prompt ולמה הוא קובע אם תקבלו תשובה מעולה או זבל
- 5 עקרונות הזהב לכתיבת Prompt טוב
- דוגמאות מעשיות שתוכלו להעתיק ולהשתמש מיד

---

## 📖 למה "לדבר עם AI" זו אומנות?

בואו נתחיל עם משל:

דמיינו שנכנסתם למסעדה ואמרתם למלצר: **"תביא לי אוכל."**

מה תקבלו? מי יודע. אולי פיצה, אולי סלט, אולי דג. המלצר יביא *משהו*, אבל סביר שזה לא בדיוק מה שרציתם.

עכשיו דמיינו שאמרתם: **"אני רוצה סטייק מדיום-רר, עם תפוחי אדמה אפויים, רוטב פלפלים, בלי בצל, ותביא גם כוס יין אדום."**

הרבה יותר טוב, נכון?

**ככה זה עם AI.** מה שאתם כותבים לו (ה-**Prompt**) קובע מה תקבלו בחזרה. Prompt מעורפל → תשובה מעורפלת. Prompt מדויק → תשובה מדויקת.

---

## מה זה Prompt?

**Prompt (פרומפט)** = הטקסט שאתם כותבים ל-AI. זו ההוראה, השאלה, הבקשה - כל מה שאתם מקלידים בתיבת הטקסט.

דוגמאות ל-Prompts:
- "מה מזג האוויר?" ← Prompt פשוט
- "תכתוב לי מייל" ← Prompt מעורפל
- "תכתוב לי מייל מקצועי בעברית למנהלת שלי, שמבקש יום חופש ביום חמישי הקרוב, בטון מנומס אבל לא נחנפני" ← Prompt מעולה

**הכלל הפשוט:** ככל שתתנו ל-AI יותר מידע על מה שאתם רוצים, כך התשובה תהיה יותר טובה.

---

## 5 עקרונות הזהב לכתיבת Prompt

### עיקרון #1: היו ספציפיים 🎯

| ❌ Prompt חלש | ✅ Prompt חזק |
|--------------|-------------|
| "תכתוב לי משהו על שיווק" | "תכתוב 5 רעיונות לפוסטים באינסטגרם לחנות פרחים קטנה בתל אביב, שמתמקדים בעונת האביב" |
| "תעזור לי עם מייל" | "תכתוב מייל קצר (5-6 שורות) בעברית, לספק שלא סיפק הזמנה בזמן, בטון מקצועי אבל תקיף" |
| "תספר לי על בישול" | "תסביר איך להכין חומוס ביתי ממש טוב, שלב אחרי שלב, למתחיל שמעולם לא בישל" |

### עיקרון #2: תנו הקשר 📋

ה-AI לא יודע מי אתם, מה אתם עושים, ולמה אתם שואלים. ספרו לו!

**❌ בלי הקשר:**
```
תכתוב לי ביו לאינסטגרם
```

**✅ עם הקשר:**
```
אני מעצבת גרפית פרילנסרית מחיפה, מתמחה בעיצוב לוגואים ומיתוג לעסקים קטנים.
קהל היעד שלי הוא בעלי עסקים קטנים בגילאי 30-50.
תכתוב לי ביו לאינסטגרם שהוא מקצועי אבל חמים, עד 150 תווים.
```

### עיקרון #3: הגדירו פורמט 📐

אמרו ל-AI באיזה צורה אתם רוצים את התשובה:

```
תענה לי ב:
- רשימת נקודות (לא פסקאות ארוכות)
- מקסימום 5 נקודות
- כל נקודה - משפט אחד
```

או:
```
תענה בפורמט של טבלה עם שלוש עמודות: פעולה, זמן, עלות
```

או:
```
תענה כאילו אתה מסביר לילד בן 8
```

### עיקרון #4: תנו דוגמאות 🎨

זו אחת הטכניקות הכי חזקות. אם אתם רוצים תוצאה בסגנון מסוים - תראו ל-AI דוגמה.

```
תכתוב לי תיאורי מוצר לחנות אונליין בסגנון הזה:

דוגמה: "הכובע הזה? הוא לא סתם כובע. הוא ההצהרה שלך לעולם שאתה מגיע.
100% כותנה. 100% סטייל. 0% התנצלויות."

עכשיו תכתוב בסגנון הזה עבור:
1. חולצת טי שחורה
2. תיק גב לעבודה
3. בקבוק מים תרמי
```

### עיקרון #5: חזרו ושפרו 🔄

לא קיבלתם מה שרציתם? אל תתחילו מאפס. **המשיכו את השיחה:**

```
שיחה לדוגמה:

אתם: "תכתוב לי מייל לספק"
AI: [כותב מייל ארוך ורשמי מדי]
אתם: "טוב, אבל זה ארוך מדי. תקצר לחצי ותהיה יותר ישיר"
AI: [כותב גרסה קצרה יותר]
אתם: "מעולה, רק תוסיף תאריך יעד לתשובה"
AI: [מוסיף]
```

**AI הוא כמו עובד מוכשר שצריך הכוונה.** אם התוצאה הראשונה לא מושלמת - זה נורמלי. שפרו, כוונו, חזרו.

## הנוסחה הסודית לכל Prompt מושלם

הנה תבנית שעובדת כמעט תמיד:

```
[תפקיד] + [משימה] + [הקשר] + [פורמט] + [מגבלות]
```

**דוגמה מפורקת:**

```
אתה [כותב תוכן מנוסה בעברית].                    ← תפקיד
תכתוב [פוסט לפייסבוק שמקדם סדנת בישול].          ← משימה
הסדנה [מיועדת לזוגות, עולה 250 ש"ח, ביום שישי     ← הקשר
הקרוב בתל אביב, מתמקדת במטבח איטלקי].
הפוסט צריך להיות [3-4 שורות, עם אימוג'י,           ← פורמט
כולל קריאה לפעולה].
[אל תשתמש במילים "הזדמנות" או "פעם בחיים"].       ← מגבלות
```

## טכניקות בסיסיות שכדאי להכיר

### 1. Zero-shot (ירי מאפס) 🎯
פשוט שואלים, בלי דוגמאות:
```
תרגם לאנגלית: "אני רוצה להזמין שולחן לשניים לערב"
```

### 2. Few-shot (כמה דוגמאות) 🎯🎯🎯
נותנים דוגמאות לפני הבקשה:
```
תרגם לאנגלית עממית (לא פורמלית):
עברית: "מה קורה אחי?" → אנגלית: "What's up bro?"
עברית: "זה היה מטורף" → אנגלית: "That was insane"
עברית: "אני מת על הפיצה הזו" → אנגלית: ???
```

### 3. שיטת ה"תפקיד" (Role Playing) 🎭
```
אתה מורה לכלכלה בתיכון. תסביר לי מה זה אינפלציה
בשפה פשוטה עם דוגמאות מחיי היומיום בישראל.
```

### 4. שיטת ה"שלב אחרי שלב" (Step by Step) 👣
```
אני רוצה לפתוח ערוץ יוטיוב.
תתן לי תוכנית פעולה שלב אחרי שלב:
1. מה לעשות ביום הראשון
2. מה לעשות בשבוע הראשון
3. מה לעשות בחודש הראשון
```

---

## 💡 דוגמאות מעשיות - Prompts מוכנים לשימוש

### Prompt #1: עזרה בכתיבת קורות חיים
```
אני רוצה לשפר את קורות החיים שלי.
אני [גננת/מתכנת/רואה חשבון/מנהל פרויקטים] עם [X] שנות ניסיון.
בבקשה:
1. תציע לי 3 משפטים חזקים לפסקת הפתיחה (Summary)
2. תעזור לי לנסח את הניסיון שלי ב-[שם חברה] בצורה שמדגישה הישגים ולא תפקידים
3. תשתמש במילות פעולה חזקות
```

### Prompt #2: תכנון שבועי
```
תכין לי תפריט ארוחות לשבוע:
- למשפחה של 4 (2 מבוגרים, 2 ילדים גילאי 6 ו-9)
- תקציב: בינוני
- מגבלות: בלי פירות ים, הילד הקטן לא אוהב פטריות
- ארוחת ערב בלבד (ארוחה ביתית רגילה, לא סושי)
- תוסיף רשימת קניות לסופר לכל השבוע
```

### Prompt #3: הכנה לראיון עבודה
```
אני הולך/ת לראיון עבודה לתפקיד [שם התפקיד] ב[סוג החברה].
1. תן לי 10 שאלות נפוצות שסביר שישאלו
2. לכל שאלה - תן טיפ קצר איך לענות
3. תן לי 3 שאלות חכמות שאני יכול/ה לשאול את המראיין
```

### Prompt #4: הסבר מושג מורכב
```
תסביר לי מה זה [בלוקצ'יין / NFT / מניות / ביטוח חיים]
כאילו אני בן 12. בלי מילים מקצועיות.
תשתמש בדוגמה מחיי היומיום.
מקסימום 10 שורות.
```

### Prompt #5: ביקורת על טקסט
```
הנה טקסט שכתבתי [הדביקו את הטקסט].
בבקשה:
1. ציין 3 דברים טובים בטקסט
2. ציין 3 דברים שאפשר לשפר
3. תשכתב את הפסקה הראשונה כדוגמה לשיפור
אל תשנה את הסגנון שלי - רק שפר.
```

## 🧪 תרגיל לבית

**חלק א': הנוסחה**
קחו את הנוסחה הסודית:
```
[תפקיד] + [משימה] + [הקשר] + [פורמט] + [מגבלות]
```
וכתבו Prompt אחד שלכם, על נושא שמעניין אתכם. נסו אותו בכלי AI.

**חלק ב': לפני ואחרי**
נסו את שני ה-Prompts האלה והשוו:

**Prompt חלש:**
```
תכתוב לי על כלבים
```

**Prompt חזק:**
```
אני שוקל לאמץ כלב לראשונה. אני גר בדירה בקומה 3 בלי מעלית,
עובד 8 שעות מחוץ לבית, ויש לי ילד בן 4.
תמליץ על 3 גזעים מתאימים ותסביר למה כל אחד מהם מתאים למצב שלי.
```

שימו לב כמה התשובה השנייה יותר שימושית!

## 🎮 שאלה לבדיקה עצמית

**מה ההבדל בין Zero-shot ל-Few-shot?**

א. Zero-shot זה חינם ו-Few-shot עולה כסף

ב. Zero-shot זה שואלים ישר, Few-shot זה נותנים דוגמאות לפני 🎯

ג. Zero-shot זה לכתיבה ו-Few-shot זה לתמונות

ד. אין הבדל, רק השם שונה

<details>
<summary>לחצו לתשובה</summary>

**התשובה הנכונה: ב'** ✅

**Zero-shot** = שואלים את ה-AI ישירות בלי דוגמאות ("תרגם: שלום" → "Hello")
**Few-shot** = נותנים כמה דוגמאות לפני הבקשה כדי שה-AI יבין את הסגנון שאנחנו רוצים. זו טכניקה חזקה מאוד כשרוצים תוצאה בסגנון מאוד ספציפי.

</details>

## 📌 סיכום הנקודות המרכזיות

- **Prompt = ההוראה שלכם ל-AI.** ככל שהוא יותר מדויק, התשובה יותר טובה
- **5 עקרונות הזהב:** היו ספציפיים, תנו הקשר, הגדירו פורמט, תנו דוגמאות, חזרו ושפרו
- **הנוסחה:** תפקיד + משימה + הקשר + פורמט + מגבלות
- **טכניקות בסיסיות:** Zero-shot, Few-shot, תפקידים, שלב-אחרי-שלב
- **אל תוותרו בקלות** - אם התשובה לא מושלמת, שפרו את ה-Prompt
