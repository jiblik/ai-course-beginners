## 📋 נוסחת ה-Prompt המושלם

```
[תפקיד] + [משימה] + [הקשר] + [פורמט] + [מגבלות]
```

**דוגמה:**
```
אתה [כותב תוכן מנוסה].                    ← תפקיד
תכתוב [מייל ללקוח].                        ← משימה
הלקוח [לא קיבל הזמנה, ממתין שבוע].        ← הקשר
המייל צריך להיות [5 שורות, מנומס, בעברית]. ← פורמט
[אל תציע פיצוי כספי].                      ← מגבלות
```

## 🔧 טכניקות מתקדמות - תזכורת

| טכניקה | מתי להשתמש | איך |
|--------|-----------|-----|
| **Zero-shot** | שאלות פשוטות | שואלים ישר |
| **Few-shot** | רוצים סגנון ספציפי | נותנים 2-3 דוגמאות לפני |
| **Chain of Thought** | חישובים/לוגיקה | "חשוב שלב אחרי שלב" |
| **Role Playing** | רוצים נקודת מבט | "אתה מורה/רופא/שף..." |
| **ביקורת עצמית** | רוצים תוצאה מושלמת | "אחרי שתכתוב, בקר את עצמך" |

## 📧 תבניות Prompt מוכנות

### מייל מקצועי
```
תכתוב מייל [סוג] ל[נמען] על [נושא].
טון: [מנומס/תקיף/ידידותי]. אורך: [X שורות].
מטרה: [מה אני רוצה שיקרה].
```

### סיכום מסמך
```
סכם את המסמך הזה:
- תקציר (3 שורות)
- נקודות עיקריות (5 bullets)
- דברים שדורשים תשומת לב
```

### סיעור מוחות
```
תן 10 רעיונות ל[נושא]:
3 בטוחים, 3 יצירתיים, 3 מטורפים, 1 שאף אחד לא חשב עליו.
```

### לימוד נושא חדש
```
לימד אותי [נושא] מאפס.
אחרי כל הסבר שאל שאלה לבדוק שהבנתי.
```

### תכנון
```
תכנן לי [משימה/אירוע/פרויקט].
פרטים: [X]. תקציב: [X]. זמן: [X].
תפורמט כלוח זמנים.
```

## ⚠️ כללי בטיחות - תזכורת

### לא לשתף:
- ❌ ת.ז., כרטיס אשראי, סיסמאות
- ❌ מסמכים סודיים של העבודה
- ❌ מידע רפואי רגיש
- ❌ פרטים אישיים של אנשים אחרים

### תמיד:
- ✅ לבדוק עובדות חשובות
- ✅ לקרוא לפני שליחה
- ✅ להתייעץ עם מומחה בהחלטות גדולות
- ✅ לכבות היסטוריית אימון (בהגדרות)
